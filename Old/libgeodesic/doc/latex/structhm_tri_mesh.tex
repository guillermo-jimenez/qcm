\subsubsection{Detailed Description}
Triangle mesh data structure. 

Represents a collection of triangles and their vertex locations via a simple vertex-\/face adjacency structure.

$L^2$ distance between two scalar functions at vertices. 

For two functions $\phi_1$ and $\phi_2$, returns the quantity $\left(\int_M (\phi_1-\phi_2)^2 \right)^{1/2}$. For piecewise linear functions this quantity can be computed exactly as $\left(\sum_{i=1}^V \mathcal{A}_i (\phi_1^i-\phi_2^i)^2 \right)^{1/2}$ where $V$ is the number of vertices, $\phi_j^i$ is the function value of the $j$th function at the $i$th vertex, and $\mathcal{A}_i$ is the barycentric dual area associated with the $i$th vertex.
