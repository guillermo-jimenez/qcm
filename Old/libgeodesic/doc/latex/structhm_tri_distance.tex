\subsubsection{Detailed Description}
Distance function on a triangle mesh. 

Represents the geodesic distance function $\phi: M \rightarrow \mathbb{R}$ to a given set $\gamma$ on a surface $M$. The domain $M$ is represented by a \hyperlink{structhm_tri_mesh}{hm\-Tri\-Mesh} -- note that no guarantees can be made about the solution unless this mesh is {\itshape manifold} (i.\-e., every edge contained in at most two triangles; every vertex contained in a single ring of triangles) and {\itshape oriented} (i.\-e., all triangles specified in a consistent winding order -- either clockwise or counter-\/clockwise). The source set is specified by setting values in \hyperlink{structhm_tri_distance_a5d82697941d1a917e535f65a6a7505c9}{hm\-Tri\-Distance\-::is\-Source}. Once the distance function has been initialized, it can be rapidly (re-\/)computed for a new source set $\gamma$ by calling \hyperlink{structhm_tri_distance_aa5107690159127c6715c85fd5847871b}{hm\-Tri\-Distance\-Update()}; the solution is stored in \hyperlink{structhm_tri_distance_a6a9fc819eb7899fe7efd9203a89ce059}{hm\-Tri\-Distance\-::distance}. Distance is computed using the method described in Crane, Weischedel, and Wardetzky, {\itshape Geodesics in Heat}, \href{http://arxiv.org/abs/1204.6216}{\tt ar\-Xiv\-:1204.\-6216v1} (2012).

Specifies boundary conditions. 

Boundary conditions are specified via a parameter in the range $[0,1]$. A value of zero yields pure Neumann conditions; a value of one yields pure Dirichlet conditions. Values in between interpolate between the two boundary conditions. For most purposes (namely\-: when the surface has no boundary or when an accurate approximation of geodesic distance is desired) this parameter has little effect and optimal performance will be achieved by simply setting it to zero. For surfaces with boundary and large heat flow time (i.\-e., smoothed geodesic distance) a value of 1/2 tends to produce natural-\/looking behavior near the boundary. {\bfseries Note\-:} in order for this parameter to take effect, its value must be set {\itshape before} calling \hyperlink{structhm_tri_distance_a681250be3575802920a769e15cd6b102}{hm\-Tri\-Distance\-Build()}.
