Detailed Description

Distance function on a triangle mesh.

Represents the geodesic distance function \(\phi: M \rightarrow \mathbb{R}\) to a given set \(\gamma\) on a surface \(M\). The domain \(M\) is represented by a hmTriMesh – note that no guarantees can be made about the solution unless this mesh is manifold (i.e., every edge contained in at most two triangles; every vertex contained in a single ring of triangles) and oriented (i.e., all triangles specified in a consistent winding order – either clockwise or counter-clockwise). The source set is specified by setting values in hmTriDistance::isSource. Once the distance function has been initialized, it can be rapidly (re-)computed for a new source set \(\gamma\) by calling hmTriDistanceUpdate(); the solution is stored in hmTriDistance::distance. Distance is computed using the method described in Crane, Weischedel, and Wardetzky, Geodesics in Heat, arXiv:1204.6216v1 (2012).

Estimates a good value for the duration of heat flow.

Uses the simple estimate \(t = A/F\) where \(A\) is the total surface area and \(F\) is the number of triangles in the mesh. The resulting value is stored in hmTriDistance::time.

void hmTriDistanceSetBoundaryConditions ( hmTriDistance *  distance,
double  boundaryConditions 
)

Specifies boundary conditions.

Boundary conditions are specified via a parameter in the range \([0,1]\). A value of zero yields pure Neumann conditions; a value of one yields pure Dirichlet conditions. Values in between interpolate between the two boundary conditions. For most purposes (namely: when the surface has no boundary or when an accurate approximation of geodesic distance is desired) this parameter has little effect and optimal performance will be achieved by simply setting it to zero. For surfaces with boundary and large heat flow time (i.e., smoothed geodesic distance) a value of 1/2 tends to produce natural-looking behavior near the boundary. Note: in order for this parameter to take effect, its value must be set before calling hmTriDistanceBuild().
